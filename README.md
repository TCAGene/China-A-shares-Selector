# China-A-shares-Selector

1. run db.py first to generate a local database file.

2. run server.py to open the local page.

3. click "更新数据“ to update the data to latest date. First run could take a long time. Afterwards the data will be updated incrementally, which will be a lot faster.

4. click"显示当前结果“ to show the selected result, ranked by their performances based on last week. Each day newly added ones will be listed on top and marked, which should be most focused on
